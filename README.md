## Setup

Criando ambiente virtual

```
python3 -m venv .venv
```

Ativando ambiente virtual:
```
source .venv/bin/activate
```

Saindo do ambiente virtual:
```
deactivate
```

Instalar pacotes no ambiente virtual:
```
source .venv/bin/activate
pip install numpy
```

Help para np.random
```
help(csv.dictReader)
dir(np.random)
```
